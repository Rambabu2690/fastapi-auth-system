# FastAPI Authentication System

This project implements a complete user authentication and management system using FastAPI and MySQL.

## Features
- User signup with password hashing and welcome email
- User login with JWT authentication
- Forgot password with OTP sent via email
- Secure password reset with OTP validation
- List users with pagination
- Update user details
- Positive and negative API test cases using Postman

## Tech Stack
- FastAPI
- MySQL
- SQLAlchemy
- JWT
- SMTP (Email)
- Postman
